# 🌐 YeobaekHub

AI 어시스턴트와 함께하는 개발자 커뮤니티 포털  
초보자도 실행할 수 있도록 친절한 설명서와 함께합니다 😊

---

## 📁 프로젝트 소개

YeobaekHub는 AI 챗봇, 연구실 소개, 프로젝트 현황, 기술 스택 안내 등의 기능을 제공하는 인터랙티브 웹 포털입니다.  
React 기반으로 만들어졌으며, 누구나 쉽게 클론 후 실행해볼 수 있도록 구성되어 있습니다.

---
